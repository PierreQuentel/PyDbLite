 PyDbLite est :
  • un moteur de base de données en mémoire, rapide, non typée, en pur Python, compatible avec Python 2.3 et au-dessus, qui utilise la syntaxe de Python pour toutes les opérations de gestion de la base au lieu de SQL. Il consiste en un petit module, PyDbLite.py
  • une interface pythonique à SQLite et à MySQL, avec la même syntaxe que le moteur en pur Python pour la plupart des opérations (à l'exception de la connection à la base et de la création de tables, à cause des spécificités de ces bases de données)

Pour l'installer, il suffit de le télécharger et de l'installer par la commande >python setup.py install
