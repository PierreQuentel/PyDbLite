 PyDbLite is
  • a fast, pure-Python, untyped, in-memory database engine, compatible with Python 2.3 and above, using Python syntax to manage data, instead of SQL
  • a pythonic interface to SQLite and MySQL, using the same syntax as the pure-Python engine for most operations (except database connection and table creation because of each database specificities)

To install the package, just download it and install it by running >python setup.py install
